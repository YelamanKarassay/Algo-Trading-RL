# Research Sweep Experiments

This folder is generated by `apps.research_sweep`. The YAML files are intentionally
configuration-only: they combine assets, candle intervals, reward functions, state encoders,
and action spaces without adding new training code.

Suggested flow:

```bash
python -m apps.research_sweep download --start 2018-01-01
python -m apps.research_sweep generate --preset core
python -m apps.research_sweep run --config-dir experiments/research
python -m apps.research_sweep report --glob "results/research_*"
```

Use `--preset full` for the exhaustive grid. It is much larger than the default core sweep
and is best run overnight.

For 1-minute data, prefer a recent window first to avoid Webull throttling and very large
training runs:

```bash
python -m apps.research_sweep download --start 2026-01-01 --interval 1m --sleep-seconds 2.0
```

The generated `1m` configs use the M1 files, but strict validation means sparse leveraged
ETFs may have too few complete days until a resampling/fill policy is added.
